\href{https://www.tablesgenerator.com/}{This https://www.tablesgenerator.com/} site is a great tool for generating latex tables.

Here, \cref{table:sample} is a multirow, 
multicolumn merged cells table. You can visit that site, use your data and bold, center, give border- infact can do any kind of styling on your table. Remember, sometimes it may show in the generated code that you need to include \begin{verbatim}{\usepackage{multirow}}\end{verbatim} in preamble. You don't need to do so- we have done it already. \textbf{Remember, in tables, we give captions above them, unlike the figures.}

\section{Text paragraph in table}
Use p{width} instead of l/c/r in tabel environment to get wrapped text. \Cref{table:nonpar} shows a nonwrapped text containing table and \Cref{table:par} shows a wrapped text containing one.

\begin{table}[!htp]
\centering
\caption{Table with no paragraph}
\begin{tabular}{|l|l|}
\hline
A quick brown fox jumps over the lazy dog & A lion does not bother itself with a sheep's opinion \\ \hline
\end{tabular}
\label{table:nonpar}
\end{table}

\begin{table}[!htp]
\centering
\caption{Table with paragraph}
\begin{tabular}{|p{2cm}|p{2cm}|}
\hline
A quick brown fox jumps over the lazy dog & A lion does not bother itself with a sheep's opinion \\ \hline
\end{tabular}
\label{table:par}
\end{table}

\section{Long table}
[We took help from \href{http://users.sdsc.edu/~ssmallen/latex/longtable.html}{http://users.sdsc.edu/~ssmallen/latex/longtable.html} for this example.]
The longtable package can help you out for showing a table spanning multiple pages. It allows you to
\begin{enumerate}
    \item specify the column headings such that it prints on each page.
    \item add a caption on each continued page to indicate that it the table is continued from the previous page.
    \item you can add a footer to indicate that a table will be continued on the following page
\end{enumerate}
 The longtable syntax is identical to the regular table environment. The  \Cref{table:long} spans more than one page:
